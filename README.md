# Task Management System (FlowTrack)

## 📌 Project Overview

FlowTrack is a task and project management web application designed to help teams organize their work efficiently. It allows administrators, managers, and employees to collaborate by creating projects, assigning tasks, and tracking progress.

## 🚀 Features

* User authentication and role-based access (Admin, Manager, Employee)
* Project creation and management
* Task assignment and tracking
* Manager dashboard for team management
* Employee dashboard to view assigned tasks
* Progress tracking and productivity monitoring
* Attendance and leave tracking
* Report generation for tasks and project performance

## 🛠 Tech Stack

Frontend:

* React.js
* CSS

Backend:

* Node.js
* Express.js
* MongoDB

## ⚙️ Installation

1. Clone the repository

git clone https://github.com/dev8116/Task-Management.git

2. Install frontend dependencies

cd flowtrack-frontend
npm install
npm start

3. Install backend dependencies

cd flowtrack-backend
npm run dev
